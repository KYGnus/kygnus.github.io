Portable Android Forensic Analysis

Poirot is a cross-platform forensic and diagnostics tool with a Flask-based web UI that enables seamless interaction with Android devices using ADB (Android Debug Bridge). It provides a secure and intuitive interface to inspect, manage, and analyze system behavior and applications.

Designed for both Linux and Windows, Poirot runs directly from a USB disk without requiring installation. This makes it ideal for forensic analysts, IT administrators, and power users who need a portable and powerful toolset for Android device analysis and management.
